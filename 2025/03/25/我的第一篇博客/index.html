从零开始学习Hexo博客搭建

​ 零基础学习博客搭建，尽管目前的博客搭建方式与平台众多，但出于对学习投入成本的考量（(´;ω;`)主要还是资金不足bushi），选择博客平台时选择了完全免费的hexo+github搭建方案。接下来我将以我的实际学习过程记录博客搭建、功能添加与完善以及学习时所遇到的问题与心得体会，希望可以一直坚持下去(ゝ∀･)

仓库搭建

​ 提前注册一个github账户并登录，进入主页后点击Create a new repository进入新建仓库页面。此时需要在Owner标题下创建你的仓库名称，注意这里必须使用注册时填写的用户名.github.io格式来填写。之后向下滑动，勾选Public,勾选Add a README file，最后点击Create repository完成创建
